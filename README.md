# Dove --- devenv as a vagrant wrapper
## Dependencies
- Docker
- Vagrant
- Virtualbox

## Installation
```
git clone https://github.com/nabinno/dove
PATH=dove/bin:$PATH
```

## Usage
Vagrant up:
```
dove up centos
ssh centos
```
Help for dove:
```
dove
```
Help for vagrant:
```
dove -h
```
